### 두 번째 과제 ( 겨우 제출 )

#### tag 입력 실수
아래 코드는 어떤 결과를 보여줄까요 ? 
``` html
<li> heoyu </li>
<li> hamzzi <li>
<li> alphaka <li>
<li> choi <li>
```
- tag 가 제대로 닫히지 않았는데 .. ! 신기했습니다.
- `li tag 는 <>인 tag 이기 때문에 그렇다.` 같은 문장처럼, 이유가 궁금한데 아직 정답을 찾지 못했습니다.
